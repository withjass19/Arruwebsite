15. Program to count the total number of punctuation characters exists in a String

Code


public class PunctuationCountExample {
    public static void main(String[] args) {
        // Input string
       String inputString = "Hello, world! This is a sample string with some punctuation.";

        // Count the number of punctuation characters
       int punctuationCount = countPunctuation(inputString);

        // Print the output
        System.out.println("Input String: " + inputString);
        System.out.println("Number of Punctuation Characters: " + punctuationCount);
   }
    // Method to count the number of punctuation characters in a string
    private static int countPunctuation(String input) {
        int count = 0;
        for (int i = 0; i < input.length(); i++) {
           char ch = input.charAt(i);
            // Using the Character class to check if a character is a punctuation character
            if (Character.isPunctuation(ch)) {
                count++;
            }
        }
        return count;
   }
}

Output

Input String: Hello, world! This is a sample string with some punctuation.
Number of Punctuation Characters: 6
