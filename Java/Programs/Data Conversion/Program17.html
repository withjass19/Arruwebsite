17. How to convert int to double

Code


public class IntToDoubleConversion {
    public static void main(String[] args) {
        // Convert int to double implicitly
        int intValue = 42;
        double doubleValue = intValue;

        // Print the results
        System.out.println("Original int value: " + intValue);
        System.out.println("Converted to double value: " + doubleValue);

        // Example of explicit casting from double to int
        double anotherDoubleValue = 78.56;
        int anotherIntValue = (int) anotherDoubleValue;

        // Print the results
        System.out.println("\nOriginal double value: " + anotherDoubleValue);
        System.out.println("Converted to int value (with explicit casting): " + anotherIntValue);
    }
}

Output

Original int value: 42
Converted to double value: 42.0

Original double value: 78.56
Converted to int value (with explicit casting): 78
